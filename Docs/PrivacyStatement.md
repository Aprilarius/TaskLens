# TaskLens Privacy Statement

TaskLens performs its analysis locally. It has no telemetry, registration, cloud account, upload or online translation feature. Process metadata and resource samples remain in memory unless the user explicitly generates a report. Settings are stored locally under `%LOCALAPPDATA%\TaskLens`; reports are stored under `Documents\TaskLens Reports`.

TaskLens does not read personal file contents, passwords or browser history. It reads executable metadata only to explain running processes and startup entries. Reports are never transmitted by TaskLens.
