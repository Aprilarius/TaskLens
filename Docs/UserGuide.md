# TaskLens User Guide

Start the app and select **Start analysis**. The Dashboard summarizes the likely reason for current slowness. Open **Processes** to search, filter and sort; double-click a row for an explanation and safe actions. **Review indicators** are cautious prompts, not malware verdicts. **Startup review** explains programs that run at sign-in without disabling them. **Reports** creates local HTML, JSON or CSV files; choose anonymized mode before sharing a report. Change language, theme and refresh interval in **Settings**.

Never end a process when you have unsaved work. TaskLens blocks its known critical Windows process list, but ordinary applications can also lose unsaved data when closed.
